```
│▓▓▓▓▓▓▓┌───────────────┐▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│

└─────────────────────────────────────────┘
         ↕ размеры показываются рядом с курсором
```

**Элементы overlay:**
- `Rectangle` поверх всего окна: `color: Qt.rgba(0,0,0,0.3)`, `z: 999`
- Внутри: `Item` с `DragHandler` для рисования прямоугольника
- Прямоугольник выделения: `border.width: 2`, `border.color: Theme.accent`, `color: "transparent"`
- Текст размеров: `Text` рядом с нижним правым углом
- Полосы-маркиры по углам (4 маленьких L-образных прямоугольника)

**Управление:**
- `Enter` / `клик по иконке подтверждения` → захват области → сохранение
- `Escape` → отмена
- Ресайз через 8 handles по углам и сторонам

## 4. Picture-in-Picture — Детали UI

### 4.1 Обнаружение видео

**Инжектируемый скрипт (на каждую страницу):**
```javascript
(function() {
    const videos = document.querySelectorAll('video');
    if (videos.length > 0) {
        document.documentElement.setAttribute('data-filka-pip', 'true');
    }
    // MutationObserver для динамических видео
    const obs = new MutationObserver(() => {
        const v = document.querySelectorAll('video');
        document.documentElement.setAttribute('data-filka-pip', v.length > 0 ? 'true' : 'false');
    });
    obs.observe(document.body, { childList: true, subtree: true });
})();
```

### 4.2 Кнопка в WebContextMenu

**Характеристики:**
- `Window { flags: Qt.WindowStaysOnTopHint | Qt.Tool }`
- Размер: 320×180 (min: 200×112, max: 1920×1080)
- Позиция по умолчанию: правый нижний угол (с отступом 20px)
- Перетаскивание: `DragHandler` по верхней части окна (30px)
- Ресайз: `DragHandler` по нижнему правому углу

**Контролы внизу окна:**
```
┌────────────────────────────────────┐
│                                    │
│         Видео контент              │
│                                    │
├────────────────────────────────────┤
│ [▶/⏸] [🔊/🔇]     [← вкладку] [✕] │
└────────────────────────────────────┘
```

| Кнопка | Иконка | Действие |
|--------|--------|----------|
| Play/Pause | play / pause | `video.play()` / `video.pause()` |
| Mute/Unmute | volume-2 / volume-x | `video.muted = !video.muted` |
| Return | maximize-2 | Закрыть PiP, вернуть видео в вкладку |
| Close | x | Закрыть PiP окно |

### После:
```
[←][→][⟳][⌂] [===== AddressBar =====] [🔖][95%][📷][🗂][🎵][🔀][🕐][⬇][🌐][🌙][⚙]
                                                 ↑   ↑   ↑   ↑
                                           скриншот  вкладки  тема  медиа
```

### Правила сдвига:
1. Если `zoomFactor === 1.0` → зум-пилл скрыт → все кнопки правее него сдвигаются влево на 36px
2. Если `audibleTabs.length === 0` → кнопка music скрыта → кнопки правее неё сдвигаются влево на 36px
3. Итого ширина тулбара может меняться от ~480px (всё скрыто) до ~630px (всё видимо)
