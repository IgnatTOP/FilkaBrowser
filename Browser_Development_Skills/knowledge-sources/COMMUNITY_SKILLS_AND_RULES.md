# Community Skills And Rules

Эти источники полезны как каталоги пользовательских rules, skills, commands и agent workflows. Для Qt-браузера их не нужно копировать целиком. Брать только подходящие паттерны: C++, CMake, security, testing, code review, release, UI/UX.

## Cursor

- Awesome Cursor Rules: https://github.com/PatrickJS/awesome-cursorrules
- Cursor Directory: https://cursor.directory/

Что искать:

- C++ rules
- CMake rules
- security review rules
- testing rules
- UI/UX implementation rules
- refactoring rules
